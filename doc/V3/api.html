Starts a new inventory (stocktaking).

The inventoried status of all articles will be set to 0.
Until the inventory is stopped, the inventoried status of every new or updated article will be set to 1.

Authorizations:
apiKey

Responses

/api/v3/inventories

Stops the current inventory (stocktaking).

The inventoried status of all articles will be set to -1.
Until a new inventory is started, the inventoried status of every new or updated article will be set to -1.

Authorizations:
apiKey

Responses

/api/v3/inventories
